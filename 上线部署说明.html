让别人也能玩：上线部署说明

目标

别人不需要连你的 Wi-Fi，只要打开一个公网网址，例如 https://你的游戏域名.com，就能注册、排位、匹配和手机游玩。

最快测试方案

  1. 把项目上传到 GitHub。
  2. 登录 Railway 或 Render。
  3. 导入这个 GitHub 仓库。
  4. 启动命令填：node server/index.js
  5. 环境变量设置：TOKEN_SECRET，填一段随机长字符串。
  6. 平台会给你一个公网网址，手机和电脑都能打开。

当前版本注意

现在账号和战绩保存在 data/db.json。这适合测试，不适合正式运营。正式版应该改成 PostgreSQL 数据库，否则云服务重启或换机器时数据可能丢失。

正式版本需要

  1. 公网服务器：Railway、Render、Fly.io、腾讯云、阿里云都可以。
  2. 数据库：PostgreSQL。
  3. 实时通信：建议从当前轮询升级为 WebSocket。
  4. 域名和 HTTPS：方便手机直接访问。
  5. 日志和后台：查看用户、战斗、错误。
